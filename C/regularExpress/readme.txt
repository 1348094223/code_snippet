1.       有些程序设计语言支持正则表达式，比如Java，Perl；有些则没有内置支持，比如C。请为你的编程语言加装一双支持正则表达式的翅膀。

你的正则表达式需要实现下述测试用例：

模式          匹配目标串   匹配结果

"3{2}5"     "3354"          "335"

"3{4}"      "3334"          null

"1|2|3"     "123"           "1"

"[12]+"     "1123"          "1", "11", "112"

“1*”        “11123”         "","1","11","111"

“1?         “11123”         “”,”1”

  

一个可选的测试用例如下：

模式                          匹配目标串           匹配结果

"1*([23])?|([234])*"            "111234"            "", "1","11","111","1112","11123","111234"

 

注意：

1.       * 表示任意个

2.       +表示至少一个

3.       ？表示0到一个

4.       {n}表示精确匹配n个

5.       []表示字符集合，匹配所包含的任意一个字符

6.       x|y表示匹配 x 或 y

7.       测试用例中null表示匹配不成功

3.       你的程序是否具有良好的语义结构
